# 데이터베이스 전문가 가이드: 개인 홈페이지용 DB 최소 설계

## 목표
- 필요한 데이터만 저장하고, 반드시 복구 가능한 운영 체계 확보

## DB가 필요한 경우
- 0원 시작 단계에서는 DB 없이 운영하는 것을 기본값으로 둠
- 로그인/회원 기능
- 방명록/댓글
- 관리자 글쓰기
- 문의 내역 저장

## 권장 구성
- PostgreSQL(Supabase 등) 우선
- 초기 스키마는 최소화
  - `messages`
  - `projects`
  - `guestbook_entries`

## 스키마/인덱스 원칙
- 조회 패턴 기준 최소 인덱스
- 마이그레이션 버전 관리(Prisma/Flyway)
- PII(개인정보) 최소 수집
